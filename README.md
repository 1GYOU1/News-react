### 📌 주요 기능

- react-router-dom
    - useNavigate 페이지 이동
    - useLocation 현재 페이지 파악
- styled-components 스타일 적용
- axios를 활용해 API 데이터 노출
    - Promise async/await 방식 
- 카테고리에 따른 api 노출

<br>

### 📌 환경 세팅

>$ cd news-react-app

>$ npm i react-rounter-dom

>$ npm i styled-components

>$ npm i axios
